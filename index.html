Competition Guidelines

Eligibility & Requirements
  • Open to students of all skill levels
  • Individual or team participation (max 3 members)
  • Must use only static content (HTML, CSS, JavaScript)
  • Must be deployable on GitHub Pages
Judging Criteria
  • Design & Aesthetics (40%): UI/UX, visual appeal, animations
  • Responsiveness (30%): Mobile-first approach, cross-device compatibility
  • AI Integration (20%): Creative use of AI tools for development
  • Code Quality (10%): Clean, maintainable, and well-documented code
Technical Requirements
  • Pure HTML, CSS, and JavaScript only
  • No backend dependencies or build steps
  • All content must be static
  • Must use responsive design (Flexbox/Grid)
  • Performance optimized for fast loading
